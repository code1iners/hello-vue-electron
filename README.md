# Hello Vue Electron

This is a sample application which Electron + Vue3 + Typescript + Tailwind CSS.

## Project setup

```
npm install
```

## Compiles and hot-reloads for development

```
npm run serve
npm run electron:serve
```

## Compiles and minifies for production

```
npm run build
npm run electron:build
```
